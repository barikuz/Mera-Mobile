# Mera (Mobil)

Mera, balıkçıların avlanma süreçlerinin her adımında yanlarında olan, onların işini kolaylaştıran dijital bir asistan uygulamasıdır.

## ✨ Temel Özellikler

- 📍 **Akıllı Mera Önerileri:** Bölgeye, hava koşullarına ve mevsime uygun nokta tavsiyeleri.
- 🎣 **Ekipman Tavsiyesi ve Eğitim:** Hedef balığa ve avlak koşullarına özel olta, makine ve yem önerileri. Etkili düğüm/taktik ipuçları.
- ⛅ **Hava Durumu ve Güvenlik:** Anlık hava durumu tahminleri ve gerçek zamanlı güvenlik uyarıları.
- 📊 **Av İstatistikleri:** Yakalanan balıkların konum, tür ve boyut bilgileriyle kaydedilmesi.
- 🗺️ **Keşif:** Yeni meralar için interaktif harita.
- 🛒 **Dahili Mağaza:** Temel balıkçılık malzemelerinin temin edilebileceği tek satıcılı e-ticaret bölümü.

## 🚀 Kurulum ve Çalıştırma

Bu proje Expo kullanılarak oluşturulmuştur. Geliştirme ortamını kurmak için aşağıdaki adımları izleyin:

1️⃣ Bağımlılıkları yükleyin:

```bash
npm install
```

2️⃣ Uygulamayı başlatın:

```bash
npx expo start
```

Çalıştırdıktan sonra terminaldeki yönergeleri takip ederek uygulamayı Android Emulator veya telefonunuzdaki Expo Go uygulaması üzerinden test edebilirsiniz.
